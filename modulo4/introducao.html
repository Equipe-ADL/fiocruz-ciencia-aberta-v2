Módulo 4 . Introdução Gestão, Compartilhamento, Abertura de Dados para Pesquisa

Seja bem-vindo a Gestão, Compartilhamento e Abertura de Dados para Pesquisa, o quarto módulo do Programa de Formação Modular em Ciência Aberta. Nele, você conhecerá diversos conceitos relacionados com dados para pesquisa, refletindo sobre a importância da gestão de dados para a pesquisa científica. Além disso, será apresentado a questões jurídicas relevantes sobre esse tipo de resultado da atividade de pesquisa. Desejamos a você bons estudos!
